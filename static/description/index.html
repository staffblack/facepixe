Integrate Meta Pixel to a website and track customer visits
Description

Add the Meta Pixel code to your Odoo website and get analytic data that could be used for understanding visitor's actions. With this script, you get ability to track Website Activities, Improve Your Return on Advertising, Reach New and Existing Customers on Facebook Ads.

To track the eCommerce Facebook Pixel events, such as "ViewContent", "Search", "AddToCart", "InitiateCheckout", "AddPaymentInfo", and "Purchase" on your Odoo website, use the app: eCommerce Facebook Pixel.

Configuration
Adding a tracking ID
Go to the menu "Website" - "Configuration" - "Settings" and activate the "Facebook | Meta Pixel" option. Then enter Tracking ID.
Apply the changes.
Results
The Facebook Pixel will be activated on all pages of the website.
The script sample on a webpage:
  • You will not see the Facebook Pixel script if you are logged in to Odoo and you have Website editor's access rights.
  • This behaviour is implemented to prevent sending data from internal users.
  • Log out from the system or use the incognito mode or other browser to test Pixel script.
